## Nodes
- **max7219-7seg**: Controls mx7219 7-Segment Modules.
- **max7219-mtx**: Controls mx7219 Matrix Modules.

## Installation
Install with Node-Red Palette Manager or npm command:
```
cd ~/.node-red
npm install node-red-contrib-iiot-rpi-max7219
```
